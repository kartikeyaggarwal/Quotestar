- Golang Installed

##  Installation
1. Clone The Repositry 
`git clone https://github.com/kartikeyaggarwal/Quotestar`
2. Now configure API Key in code 

> Go to main.go
> Edit  API KEY HERE WITH YOUR API KEY
API URL https://rapidapi.com/martin.svoboda/api/quotes15
